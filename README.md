# Atividade de Programação de Computadores em Linguagem C

## Descrição

Este projeto consiste em uma série de exercícios práticos desenvolvidos em Linguagem C, focados no uso de laços de repetição (`for`). O objetivo é demonstrar a manipulação de loops para interagir com o usuário e exibir informações de forma repetitiva. Cada arquivo `.c` contém um algoritmo específico, e seus respectivos executáveis estão disponíveis na pasta `output`.

## Algoritmos Incluídos

* **for1.c**: Um programa simples que utiliza um laço `for` para imprimir o nome "Thiago" 14 vezes, exibindo a contagem de cada iteração.
* **for2.c**: Este programa solicita ao usuário seu nome e um número de repetições. Em seguida, utiliza um laço `for` para exibir o nome do usuário a quantidade de vezes informada.
* **for3.c**: O programa pede o nome do usuário e um intervalo numérico (valor inicial e final). Depois, usa um laço `for` para imprimir o nome do usuário repetidamente, seguindo a contagem do intervalo especificado.
